The non-text semantic-decoy icon shortcut family (`non_text_semantic_decoy_icon`:
central colored causal icon + larger competing-class corner icon, no written words)
is a **positive supporting** second shortcut family for CIC. On real pretrained
OpenCLIP ViT-B-32 / laion2b_s34b_b79k it passed all 8 strict gates at BOTH the n=64
pilot and the n=128 scale run. Headline-supporting numbers (n=128): clean 1.000,
misleading original 0.297, oracle repair 1.000, CIC top-1 0.711, top-3 0.359,
clean-safe 0.766, matched random 0.258, CIC−random gap +0.453, clean-safe clean drop
0.008.

It is integrated as **supporting evidence only** (`semantic_decoy_eligible = true`,
`semantic_decoy_include_in_headline = false`). The primary/final headline remains the
text-overlay hard multi-decoy result (0.250 → 0.750 vs matched random 0.331, n=32
frozen, ViT-B-32). Do NOT claim universal/general shortcut generalization, open-world
discovery, cross-shortcut transfer, universal repair, or exact localization — it is a
single-model, single-family controlled oracle-intervention result.

The earlier flat [[visual-decoy-pilot-boundary]] pilot remains boundary evidence (its
one failed gate is shortcut strength, not method); the semantic-decoy icon benchmark
was the final pre-specified second-family attempt and passed all gates.
